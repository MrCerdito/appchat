Cómo instalar el widget

Copia el script y pégalo justo antes de </body> en tu HTML. El widget leerá la configuración guardada automáticamente.

HTML 
{{ scriptIntegracion }}
  1. 1
    Configura — ajusta apariencia, comportamiento y diseño del chat.
  2. 2
    Guarda — pulsa Guardar para confirmar los cambios.
  3. 3
    Pega el script — antes de </body> en tu página HTML.
  4. 4
    ¡Listo! — el widget aparece con tu configuración.
