VPS Docker - Listo para desplegar
Los Dockerfiles compilaron exitosamente. En el VPS:
# 1. Subir proyecto
rsync -avz --delete --exclude '.git' --exclude 'node_modules' --exclude '.env' \
  ./ root@<IP_VPS>:/opt/chat/

# 2. En el VPS
cd /opt/chat
cp backend/.env.production backend/.env
nano backend/.env  # completar API keys (GEMINI, RESEND, etc.)
docker compose up -d --build

# 3. Nginx en el VPS
cp deploy/nginx-host.conf /etc/nginx/sites-available/chat
ln -sf /etc/nginx/sites-available/chat /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx

# 4. SSL
certbot certonly --webroot -w /var/www/html -d innoovacloud.com -d www.innoovacloud.com
Archivos modificados
- backend/.env - Limpiado para desarrollo local (PostgreSQL en puerto 5433)
- backend/.env.production - Secrets generados para producción
Para detener el entorno local
docker compose stop   # detiene PostgreSQL
